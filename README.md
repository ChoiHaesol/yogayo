# Yogayo - 요가요 🧘‍♀️

## Yogayo?

: `요가/필라테스 수업 예약` + `운동 일지 작성` + `커뮤니티` 기능을 모두 가진 원스탑 요가/필라테스 플랫폼입니다.

## 주요 기능

- `수업 예약 기능` : (회원 측)수업 예약 및 수강권 관리 기능.
- `수업/회원 관리 기능` : (센터 측)수업 정원 관리 및 신청 회원 목록 관리 
- `수련 일기 작성` : 수업 참여 후 수업 일지(수련 일기) 작성 기능
- `커뮤니티` : 운동복 나눔, 센터 정보 공유 등의 커뮤니티 기능
